Somaiya Social: A College Social Media Platform


  • GitHub URL: Somaiya Social Repository

  • Somaiya Social is a platform designed exclusively for college students to connect, share ideas, and build meaningful collaborations. It provides a secure and user-friendly environment for fostering a vibrant online student community.


  • Features
    • Real-Time Chat: Instant messaging functionality for seamless communication.
